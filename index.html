Welcome to Blog & Handbook#

by Jeremy H. Boob

A tech blog is a website that focuses on publishing articles, news, and other content related to a specific area of technology. A technical handbook is a reference work that provides practical information on a selected subject, such as web development.

You can find here some posts and other useful stuff about microcontroller devices, web development, Python and C programming. This could be helpful if you are dabbled in these topics.

Check the Blog and then to Handbook section for further information.
